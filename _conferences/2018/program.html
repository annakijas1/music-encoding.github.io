Workshop: Introduction to MEI (part I)
Perry Roland

This workshop is designed to introduce MEI to those interested in its history, current uses, and future development. Room 6103
9:00 - 12:30
Workshop: Using Verovio
Laurent Pugin

Verovio (www.verovio.org) is an open-source library for engraving MEI scores. It can be used in a wide range of programming environments, including as a JavaScript toolkit in web-pages. Verovio output is in SVG, an XML vector graphic format that makes it possible to easily interact with the encoding of the music score.

In this workshop, you will have an overview of the Verovio project and its philosophy. You will learn about the current development status and about the latest additions. You will follow a step-by-step introduction on how to integrate Verovio in a web environment, from the most basic use case to more advanced, interactive and project specific ones.

Attendees are encouraged to bring example material / encoding they would like to render with Verovio. MEI and basic JavaScript / HTML knowledge is desirable but not require.
